# JGR reviewer-release checklist

This checklist records the remaining non-scientific tasks for preparing the
repository as a public reviewer or publication artifact.

- Confirm the final manuscript title and author list in `CITATION.cff`.
- Replace placeholder repository and DOI fields after archival release.
- Confirm the data access statement for the buoy-derived observations.
- Confirm the ECMWF data citation and retrieval/access route.
- Tag the exact repository state used for review.
- Archive the tagged release in a preservation repository.
- Ensure data and software citations appear in the manuscript References
  section, not only in the Open Research statement.
- Verify that no local absolute paths are required for the documented commands.
- Verify that no Python bytecode caches, local virtual environments, or private
  scratch files are included in the public release.
